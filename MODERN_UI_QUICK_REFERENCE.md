# 🎨 Modern UI Effects - Quick Reference Guide\n\n## Glassmorphism Classes\n```jsx\n// Standard frosted glass with blur\n<div className=\"frosted-glass border border-white/20 backdrop-blur-md\">...</div>\n\n// Darker glass for dark mode\n<div className=\"glass-dark backdrop-blur-sm\">...</div>\n\n// Secondary glass (lighter)\n<div className=\"glass-secondary backdrop-blur-sm\">...</div>\n```\n\n## Shadow System\n```jsx\n// Subtle shadow for cards\n<div className=\"shadow-soft\">...</div>\n\n// Floating elevation\n<div className=\"shadow-floating hover-lift\">...</div>\n\n// Maximum elevation for modals\n<div className=\"shadow-elevated\">...</div>\n\n// Glow effects\n<div className=\"shadow-glow\">...</div>\n<div className=\"shadow-glow-warm\">...</div>\n```\n\n## Gradient Backgrounds\n```jsx\n// Apply gradients to backgrounds\n<div className=\"gradient-primary text-white\">Primary Blue</div>\n<div className=\"gradient-soft\">Soft light Blue</div>\n<div className=\"gradient-warm\">Warm Amber</div>\n<div className=\"gradient-cool\">Cool Cyan</div>\n<div className=\"gradient-mesh\">Mesh Multi-Stop</div>\n<div className=\"gradient-danger\">Red Gradient</div>\n```\n\n## Hover Effects\n```jsx\n// Lift effect (elevates on hover)\n<button className=\"hover-lift transition-all\">Hover to Lift</button>\n\n// Glow effect\n<div className=\"hover-glow\">Adds glow on hover</div>\n\n// Scale effect (105%)\n<button className=\"hover-scale\">Hover to Scale</button>\n\n// Subtle scale (102%)\n<button className=\"hover-scale-sm\">Subtle Scale</button>\n```\n\n## Press/Active Effects\n```jsx\n// Zoom on press (mobile friendly)\n<button className=\"press-zoom\">Press me</button>\n\n// Shrink effect\n<button className=\"press-shrink\">Shrink on press</button>\n```\n\n## Entrance Animations\n```jsx\n// Fade and slide up\n<div className=\"scroll-fade-in\">Fades in</div>\n\n// Scale from small\n<div className=\"scroll-scale-in\">Scales in</div>\n\n// Slide from left\n<div className=\"scroll-slide-left\">Slides from left</div>\n\n// Slide from right\n<div className=\"scroll-slide-right\">Slides from right</div>\n\n// Slide from top\n<div className=\"scroll-slide-top\">Slides from top</div>\n\n// With stagger delay\n{items.map((item, i) => (\n  <div \n    key={item.id}\n    className=\"scroll-fade-in\"\n    style={{ animationDelay: `${i * 50}ms` }}\n  >\n    {item.name}\n  </div>\n))}\n```\n\n## Floating Animations\n```jsx\n// Gentle float (3s)\n<div className=\"animate-float\">Floating</div>\n\n// Slower float (4s)\n<div className=\"animate-float-slow\">Slow floating</div>\n```\n\n## Glow Animations\n```jsx\n// Pulsing glow\n<div className=\"pulse-glow\">Glowing</div>\n\n// Neon glow effect\n<div className=\"animate-neon-glow\">Neon effect</div>\n```\n\n## Loading States\n```jsx\n// Skeleton loader card\n<div className=\"skeleton rounded-2xl w-full h-32\">Loading...</div>\n\n// Skeleton text\n<div className=\"skeleton-text\">Loading text...</div>\n\n// Skeleton card\n<div className=\"skeleton-card\">Loading card...</div>\n```\n\n## Transition Classes\n```jsx\n// Standard smooth transition (0.3s)\n<div className=\"transition-smooth\">Smooth</div>\n\n// Fast transition (0.15s)\n<div className=\"transition-smooth-fast\">Fast</div>\n\n// Slow transition (0.5s)\n<div className=\"transition-smooth-slow\">Slow</div>\n```\n\n## Focus States (Accessibility)\n```jsx\n// Standard focus ring outline\n<button className=\"focus-ring\">Focus me</button>\n\n// Inner ring focus (for inputs)\n<input className=\"focus-inner-ring\" />\n```\n\n## Text Effects\n```jsx\n// Text with glow\n<h1 className=\"text-glow\">Glowing text</h1>\n\n// Soft shadow on text\n<p className=\"text-shadow-soft\">Shadowed text</p>\n```\n\n## Common Patterns\n\n### Modern Card\n```jsx\n<div className=\"rounded-2xl frosted-glass border border-white/20 backdrop-blur-md p-4 hover-lift shadow-soft transition-all\">\n  {/* Card content */}\n</div>\n```\n\n### Primary Button\n```jsx\n<button className=\"px-4 py-2 rounded-xl bg-gradient-primary text-white font-medium shadow-floating hover-scale press-zoom transition-all\">\n  Click me\n</button>\n```\n\n### Modern Input\n```jsx\n<input className=\"w-full h-10 px-3 rounded-xl frosted-glass backdrop-blur-sm border-white/30 text-sm focus-ring\" />\n```\n\n### Animated List\n```jsx\n<div className=\"space-y-2.5\">\n  {items.map((item, i) => (\n    <div\n      key={item.id}\n      className=\"scroll-fade-in p-4 rounded-2xl frosted-glass hover-lift shadow-soft\"\n      style={{ animationDelay: `${i * 50}ms` }}\n    >\n      {item.content}\n    </div>\n  ))}\n</div>\n```\n\n### Loading Skeleton\n```jsx\n{isLoading ? (\n  <div className=\"grid grid-cols-2 gap-3\">\n    {[1, 2, 3, 4].map(i => (\n      <div key={i} className=\"skeleton rounded-2xl h-32\" />\n    ))}\n  </div>\n) : (\n  {/* Content */}\n)}\n```\n\n## Tailwind Animation Extensions\n\nAdded to `tailwind.config.ts`:\n\n```javascript\nanimation: {\n  \"slide-in-from-bottom\": \"slide-in-from-bottom 0.4s ease-out\",\n  \"slide-in-from-top\": \"slide-in-from-top 0.4s ease-out\",\n  \"scale-in\": \"scale-in 0.3s ease-out\",\n  \"float\": \"float 3s ease-in-out infinite\",\n  \"pulse-glow\": \"pulse-glow 2s ease-in-out infinite\",\n  \"neon-glow\": \"neon-glow 2.5s ease-in-out infinite\",\n  \"shimmer\": \"shimmer 2s infinite\",\n}\n```\n\n## Dark Mode Support\n\nAll classes automatically adapt to dark mode:\n```css\n@media (prefers-color-scheme: dark) {\n  .glass { background: hsla(220, 20%, 10%, 0.5); }\n  .frosted-glass { /* Dark variant */ }\n  .skeleton { background: hsl(218, 13%, 23%); }\n}\n```\n\n## Performance Tips\n\n1. ✅ Use `transform` and `opacity` for animations (GPU accelerated)\n2. ✅ Limit backdrop-blur on very large elements\n3. ✅ Stagger animations with small delays (50-100ms)\n4. ✅ Use skeleton loaders instead of spinners\n5. ✅ Combine multiple effects for visual impact\n6. ✅ Test on mobile devices for smooth 60fps\n7. ✅ Respect `prefers-reduced-motion` for accessibility\n8. ✅ Use `will-change` sparingly for critical animations\n\n---\n\n**All effects are production-ready and tested on modern browsers!**\n
